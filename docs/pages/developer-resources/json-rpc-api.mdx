# JSON-RPC API Reference

Radius exposes a JSON-RPC 2.0 API that is compatible with standard Ethereum tooling. All methods are available over HTTP and WebSocket connections.

## Network Configuration

| Network | RPC Endpoint | Chain ID |
|---------|--------------|----------|
| Testnet | `https://rpc.testnet.radiustech.xyz` | `1223953` |

## Quick Start

```bash
curl -X POST https://rpc.testnet.radiustech.xyz \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_chainId","params":[],"id":1}'
```

Response:
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x12ad11"
}
```

## eth_* Methods

Standard Ethereum JSON-RPC methods for account queries, transactions, and state access.

### eth_chainId

Returns the chain ID of the network.

**Parameters:** None

**Returns:** `string` - The chain ID as a hex string

**Example:**
```bash

  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_chainId","params":[],"id":1}'
```

```json
{"jsonrpc":"2.0","id":1,"result":"0x12ad11"}
```

### eth_getBalance

Returns the balance of an address.

:::info
On Radius, fees are paid in stablecoins (SBC) rather than a native gas token. This method returns your actual native balance plus the maximum native token you could receive by converting your SBC through the turnstile. This supports wallets that check balance before allowing transactions.
:::

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| address | `string` | The address to query (20 bytes, hex encoded) |
| block | `string` | Block number or tag (`latest`, `earliest`, `pending`) |

**Returns:** `string` - The balance in wei as a hex string

**Example:**
```bash
curl -X POST https://rpc.testnet.radiustech.xyz \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_getBalance","params":["0x0000000000000000000000000000000000000000","latest"],"id":1}'
```

### eth_call

Executes a call without creating a transaction (dry run).

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| transaction | `object` | Transaction call object |
| block | `string` | Block number or tag (optional) |

Transaction call object:
| Field | Type | Description |
|-------|------|-------------|
| from | `string` | Sender address (optional) |
| to | `string` | Contract address |
| data | `string` | Encoded function call |
| value | `string` | Value to send (optional) |
| gas | `string` | Gas limit (optional) |

**Returns:** `string` - The return value of the executed contract call

**Example:**
```bash
# Query SBC token name
curl -X POST https://rpc.testnet.radiustech.xyz \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc":"2.0",
    "method":"eth_call",
    "params":[{
      "to":"0xF966020a30946A64B39E2e243049036367590858",
      "data":"0x06fdde03"
    },"latest"],
    "id":1
  }'
```

## Error Codes

Radius uses standard JSON-RPC 2.0 error codes:

| Code | Message | Description |
|------|---------|-------------|
| -32700 | Parse error | Invalid JSON |
| -32600 | Invalid request | Invalid request object |
| -32601 | Method not found | Method does not exist |
| -32602 | Invalid params | Invalid method parameters |
| -32603 | Internal error | Internal JSON-RPC error |
| 3 | Execution reverted | Contract execution reverted |
