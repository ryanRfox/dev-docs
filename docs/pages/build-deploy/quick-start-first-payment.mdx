# Quick Start: First Payment [Zero to first transaction on Radius in 5 minutes. Code-first, copy-paste ready.]

## Prerequisites

You'll need:
- **Node.js 18+** and **pnpm**
- A text editor (VS Code, etc.)
- Basic understanding of private keys and wallet addresses
- 2–3 minutes to follow along

<Callout type="info">
This guide uses Radius Testnet. You'll get free test tokens to experiment with.
</Callout>

## Get Test Tokens

Before sending a transaction, grab free test USD tokens from the faucet.

1. Visit the [Radius Testnet Faucet](https://testnet.radiustech.xyz/testnet/faucet)
2. Connect your wallet or paste a testnet address
3. Claim 10 USD tokens (instant)
4. Wait ~30 seconds for confirmation

<Callout type="warning">
Testnet tokens have no real value. Use them freely to learn and test.
</Callout>

## Connect to Radius

Create a project and install the Radius SDK:

```bash
mkdir radius-quickstart && cd radius-quickstart
pnpm init
```

Add `"type": "module"` to your `package.json` for ESM support:

```json
{
  "name": "radius-quickstart",
  "type": "module"
}
```

Install the SDK, viem, and TypeScript runner:

```bash
pnpm add @radiustechsystems/sdk viem tsx
```

Create a file `transaction.ts`:

```typescript
import { createRadiusClient, radiusTestnet } from '@radiustechsystems/sdk';

// Connect to Radius Testnet
const client = createRadiusClient({ chain: radiusTestnet });

// Verify connection
const chainId = await client.getChainId();
console.log('Connected to Radius Testnet, chain ID:', chainId);
```

Run it:

```bash
npx tsx transaction.ts
```

You should see: `Connected to Radius Testnet, chain ID: 1223953n`

<Callout type="info">
The `n` suffix indicates a JavaScript BigInt for precise handling of large numbers.
</Callout>

If you get a connection error, verify the RPC endpoint is correct and check your internet connection.

## Send Your First Transaction

Now let's send USD tokens. You'll need:
- A **private key** (from your testnet wallet)
- A **recipient address** (another testnet address)

<Callout type="warning">
Never commit private keys to version control. Use environment variables or `.env` files in `.gitignore`.
</Callout>

Create `.env`:

```
RADIUS_PRIVATE_KEY=0x...your_private_key_here...
RECIPIENT_ADDRESS=0x...recipient_address_here...
```

Update `transaction.ts`:

```typescript
import {
  createRadiusClient,
  createPrivateKeySigner,
  radiusTestnet,
  RadiusError,
  InsufficientBalanceError,
  type Address,
} from '@radiustechsystems/sdk';
import { parseEther } from 'viem';

// Load from environment
const privateKey = process.env.RADIUS_PRIVATE_KEY as `0x${string}`;
const recipientAddress = process.env.RECIPIENT_ADDRESS as Address;

if (!privateKey || !recipientAddress) {
  throw new Error('Missing RADIUS_PRIVATE_KEY or RECIPIENT_ADDRESS in .env');
}

// Connect to Radius Testnet
const client = createRadiusClient({ chain: radiusTestnet });

// Create signer from private key
const signer = createPrivateKeySigner(privateKey, radiusTestnet.id);

console.log('Signer created:', signer.address);

// Check balance
const balance = await client.getBalance(signer.address);
console.log('Balance:', balance.toString(), 'wei');

// Send 0.5 USD to recipient
const amount = parseEther('0.5');

console.log('Sending transaction...');

try {
  const receipt = await client.sendAndWait(signer, recipientAddress, amount);

  console.log('Transaction sent!');
  console.log('Hash:', receipt.transactionHash);
  console.log('Status:', receipt.status);  // 'success' or 'reverted'
  console.log('Block:', receipt.blockNumber);
  console.log('Gas used:', receipt.gasUsed);
} catch (error) {
  if (error instanceof InsufficientBalanceError) {
    console.error(`Insufficient funds: have ${error.balance}, need ${error.required}`);
  } else if (error instanceof RadiusError) {
    console.error(`Error: ${error.shortMessage}`);
    console.error(`Details: ${error.details}`);
  } else {
    throw error;
  }
}
```

Run it with the `.env` file:

```bash
node --env-file=.env --import=tsx transaction.ts
```

<Callout type="info">
Node.js 20+ supports `--env-file` natively. For older versions, use `dotenv` or export variables manually.
</Callout>

You should see output like:

```
Signer created: 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266
Balance: 10000000000000000000 wei
Sending transaction...
Transaction sent!
Hash: 0xabcd...
Status: success
Block: 12345n
Gas used: 21000n
```

Congratulations—you've sent your first transaction on Radius!
